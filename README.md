# API REST Java

##### Projeto API REST em Java para Vaga de Programador Junior

<br>

## Recursos necessários para a execução

* JDK 25
* MySQL 8
* GIT
* Postman

<br>

## Como executar

* Clonar o código fonte do GitHub para um repositório local
* Abrir o código fonte em alguma IDE (usei IntelliJ)
* Baixar o script "create-database-vaga-junior.sql" que está da pasta /docs e criar o banco de dados
* Baixar o script JSON "REST-API-Java.postman_collection.json" que está na pasta /docs e importar no Postman para testar os Endpoints
* Acesso ao Swagger em http://localhost:8080/swagger-ui/index.html

<br>

## Recursos utilizados no desenvolvimento
* Banco de Dados MySQL 8
* Postman para testar requisições HTTP
* Spring Framework
* Spring Boot
* Spring Data JPA
* Maven para gerenciamento de dependências
* Jakarta para persistência de dados
* Flyway para criação do banco de dados via Migrations
* Documentação da API com Swagger
